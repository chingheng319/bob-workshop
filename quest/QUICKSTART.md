# IBM Bob 知識測驗 - 快速開始指南

## 📋 專案簡介

這是一個互動式的 IBM Bob 知識測驗系統，包含 50 道精心設計的題目，涵蓋 Bob 的核心概念、企業功能、SDLC 整合等多個面向。

## 🚀 快速啟動

### 使用本地伺服器

```bash
# 使用 npx (無需安裝)
cd quest
npx http-server -p 8000 -y
```

然後在瀏覽器開啟：http://localhost:8000

## 📁 檔案結構

```
quest/
├── index.html        # 主頁面
├── questions.json    # 題庫（50 題）
├── quiz.css         # 雜誌風格樣式
├── quiz.js          # 測驗邏輯
└── QUICKSTART.md    # 本文件
```

## 🎯 功能特色

### 核心功能
- ✅ **隨機抽題**：每次點擊「隨機抽取」載入新題目
- ✅ **即時反饋**：選擇答案後立即顯示正確/錯誤
- ✅ **詳細解釋**：每題都有完整的答案解釋
- ✅ **統計追蹤**：自動記錄答題統計（使用 localStorage）

### 題目分類
1. **核心概念**（8 題）- Bob 的基本定位與價值
2. **多模式能力**（8 題）- Plan/Code/Ask 模式
3. **企業功能**（8 題）- 安全、成本、治理
4. **SDLC 整合**（8 題）- 開發生命週期支援
5. **應用場景**（8 題）- 實際應用案例
6. **技術細節**（10 題）- Slash Commands、Rules、Skills 等

## 💾 資料儲存

測驗統計會自動儲存在瀏覽器的 localStorage 中：
- 答對題數
- 答錯題數
- 總答題數
- 正確率

清除資料：開啟瀏覽器開發者工具 → Application → Local Storage → 刪除 `bob-quiz-stats`

## 📝 自訂題庫

編輯 `questions.json` 新增或修改題目：

```json
{
  "id": 51,
  "category": "核心概念",
  "question": "您的問題？",
  "options": [
    "選項 A",
    "選項 B",
    "選項 C",
    "選項 D"
  ],
  "correct": 0,
  "explanation": "答案解釋"
}
```

**注意**：
- `correct` 是索引值（0-3）
- 確保 JSON 格式正確
- 建議使用 JSON 驗證工具檢查

## 🐛 常見問題

### Q: 題目無法載入？
A: 檢查 `questions.json` 是否存在且格式正確。使用本地伺服器而非直接開啟檔案。

### Q: 統計資料消失？
A: 檢查瀏覽器是否啟用 localStorage，或是否清除了瀏覽器資料。

### Q: 樣式顯示異常？
A: 確認網路連線正常（需載入 Google Fonts），或檢查 CSS 檔案是否完整。

### Q: 如何重置測驗？
A: 目前版本需手動清除 localStorage。未來版本會加入重置按鈕。
